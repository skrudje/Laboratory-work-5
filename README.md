# Лабораторная работа №4 — Многопоточность

## Цель работы
Изучение принципов реализации многопоточности и практическое применение возможностей языков высокого уровня при моделировании бизнес-процессов.

## Методика выполнения
1. Разработка алгоритмов решения задач по индивидуальному заданию.  
2. Написание и отладка программ на языке **C++**.  
3. Тестирование программ на различных исходных данных.

---

## Задание 1 — Примитивы синхронизации

Реализован параллельный запуск заданного числа потоков, генерирующих случайные символы из ASCII-таблицы (печатаемые символы 32–126) в формате гонки.

Произведён сравнительный анализ следующих примитивов синхронизации:
- **1.a. Mutexes**  
- **1.b. Semaphore**  
- **1.c. Barrier**  
- **1.d. SpinLock**  
- **1.e. SpinWait**  
- **1.f. Monitor**

Для каждого примитива выполнены точные замеры времени выполнения с использованием `std::chrono::high_resolution_clock`.  
Общее количество операций фиксировано и равномерно распределяется между потоками, что позволяет объективно сравнить накладные расходы каждого примитива.
